Tools and links

Everything we use during the workshop, in one place.

Coding agents

Claude Code

Anthropic's agentic coding tool. Terminal-based, works with Claude models. Our primary demo tool.

OpenCode

Open-source AI coding agent. Works with any LLM provider — OpenRouter, Anthropic, OpenAI, or local models.

Model providers

OpenRouter

Unified API for dozens of models — Claude, GPT, Gemini, DeepSeek, Qwen, and more. One key, many providers. The practical way to swap models freely.

Hugging Face

Open model hub. Browse, download, and run open-weight models. Home of the Open LLM Leaderboard and inference API.

Workshop collaboration

HackMD

Collaborative Markdown editor. We use this for shared notes during the workshop — everyone can edit in real time.

tldraw

Collaborative whiteboard. Useful for sketching workflows, architectures, and pipeline diagrams during exercises.

Research tools

refine.ink

AI-powered peer review. Analyzes academic papers to identify errors in logic, mathematics, and consistency before submission.

dodo

DuckDB extension for running .do pipelines. Show SQL, assert, and dodoc — the structured IR we demo in Act 3.
