What this shows

  1. Every model got both tasks right — writing and reading Stata code. From Opus 4.8 to DeepSeek V4 Flash, the core competence is commoditized.
  2. Only Opus caught the bug — the prompt uses use to load a .csv file, which is wrong in Stata (should be import delimited). Only Opus 4.8 flagged this. Domain expertise still differentiates frontier models.
  3. Cost varies 200x for the same answer — DeepSeek V4 Flash cost 0.006¢ for the explanation task. Opus cost 1.06¢. For routine comprehension, the cheap model is indistinguishable.
  4. Reasoning tokens often waste money — Qwen spent 1,200 tokens reasoning about word count to produce a 68-word answer. Step 3.7 Flash used 957 tokens for 50 words. Sonnet 4.5 spent 56 tokens and got the writing task right. More thinking is not better thinking.
