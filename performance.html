What this shows

  1. All frontier models got the Stata right — collapse (mean) revenue, by(year) is the canonical answer, and every model above 20B parameters found it.
  2. Cost varies 100x — Sonnet 4.5 cost 0.19¢. Kimi K2.6 cost 1.01¢ for the same answer, mostly because of reasoning tokens. DeepSeek V4 Flash cost 0.01¢.
  3. Speed varies 1,700x — Mercury 2 (diffusion LLM) delivered 710 tokens/sec. Gemma 4 delivered 38. For a simple task, both are fine. For an agent loop running hundreds of calls, speed compounds.
  4. Reasoning is not always helpful — Kimi K2.6 spent 2,400 tokens deliberating whether to include use, then omitted it. Sonnet 4.5 spent 56 tokens and got it right.
